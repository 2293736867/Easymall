# 1 Easymall

一个商城项目，目前仅完成一小部分。

相关库以及技术栈：

- `Kotlin`
- `Spring Boot WebFlux`
- `Mongodb`
- `Docker`
- `BootstrapVue`
- `Vue`
- `axios`

# 2 项目目录说明

- `backend`：后端项目目录，使用`IDEA`进行开发
- `frontend`：前端项目目录，使用`WebStorm`进行开发

# 3 目前已完成

- 前端用户登录注册
