# 1 Easymall

一个前后端分离的商城项目，还处于开发阶段。

前端库：

- `ElementPlus`
- `Vue3`
- `axios`
- `vue-router`
- `vite`
- `jssha`

后端库：

- `Spring Boot WebFlux`
- `Reactive Mongodb`
- `Docker`
- `JWT`
- `guava`
- `hutool`
- `bouncycastle`
- `snakeyaml`

# 2 项目目录说明

- `backend`：后端目录
- `frontend`：前端目录
